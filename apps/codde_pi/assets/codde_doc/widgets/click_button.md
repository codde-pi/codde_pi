## Click button

[image]

### Description

The click button is the most simple button available. User tap down, button do its job.

### Datasheet

| Property  | Value        |
| --------- | ------------ |
| Size (\*) | 1x1          |
| Behavior  | user_trigger |

(\*) : arbitrary unit representing a reasonable digit dimension on the screen

### API

| user_trigger | data       |
| ------------ | ---------- |
| OnTapDown    | {#ID:true} |
| OnTapUp      | -         |
